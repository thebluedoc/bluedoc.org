优雅高效 轻松编写、分享、管理文档

BlueDoc 是一个用于解决企业内分散在各处的文档，通过统一平台的方式，将企业所有的文档汇聚在一起，形成一个便于交流、分享、协作的文档平台。

为企业私有化部署而设计的文档管理工具

凭借强大丰富的功能、直观界面，BlueDoc 可让你从最初的想法到最终文档，创造最佳工作成果

  • 私有化部署文档

    最安全的方式来为您和您的团队创建、编辑和协作文档，你可以自由选择在云平台部署或在企业内部部署

  • 团队到知识库的组织方式

    我们通过组建不同项目组（团队），划分具体项目（知识库）的方式来组织文档

  • 规范化的文档目录组织方式

    我们通过目录编辑器让文档组织、结构变得更加的规范、清晰

  • 简单快捷的文档编辑模式

    当您开始输入时，界面逐渐消失，让你沉浸式完成编写文档

  • Markdown 风格的键盘快捷键

    我们支持类似 Markdown 风格的快捷编辑方式，让熟悉 Markdown 的人能快速入手

  • 完整的文档历史记录

    在你每次发布文档以后，BlueDoc 将会为文档创建历史版本，便于某天可以用来恢复、撤销编辑

更精致易用的文档编辑、阅读排版

不同于各类文档工具，BlueDoc 是为在线编辑、阅读而设计，我们有更符合于浏览器阅读的排版

  • Markdown 导入支持

    支持 Markdown 文件打包导入，让已有内容快速迁移进去

  • PDF / Markdown 导出

    当你需要对外传播的时候，PDF 或 Markdown 打包导出能给你带来很大的帮助
